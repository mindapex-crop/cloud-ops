## 🔧 统一接入方案

### 使用 LiteLLM 统一所有免费 API
```python
# pip install litellm
import litellm

# 智谱AI
litellm.completion(model="openai/glm-4-plus", messages=[...],
                   api_base="https://open.bigmodel.cn/api/paas/v4/",
                   api_key="your-key")

# Google Gemini
litellm.completion(model="gemini/gemini-2.5-flash", messages=[...])

# Groq
litellm.completion(model="groq/llama-4-scout-17b-16e", messages=[...])

# OpenRouter
litellm.completion(model="openrouter/meta-llama/llama-4-maverick", messages=[...])
```

### 负载均衡策略
```
请求 → LiteLLM Router → 智谱AI (主力, 免费)
                     → Gemini Flash (备用, 免费)
                     → Groq (快速推理, 免费)
                     → Cloudflare Workers AI (边缘, 免费)
```
这样4个完全免费的 API 轮询使用，相当于一个不花钱的无限量 AI 后端。

## 💡 CodeWhale — 终端AI编程助手

> 来源: https://github.com/Hmbown/CodeWhale（3.7万+ stars）

CodeWhale 是一个基于 DeepSeek V4 的终端编程 Agent，使用 Rust 编写。

| 特性 | 说明 |
|------|------|
| 模型 | DeepSeek V4 Pro/Flash |
| 定价 | Pro: $0.87/1M output, Flash: $0.28/1M output |
| 自动模式 | --model auto 自动选择模型+思考深度 |
| Provider | 支持DeepSeek/NVIDIA/OpenRouter/SiliconFlow等多家 |
| LSP | 编辑后自动诊断（Rust/Python/TS/Go/C++） |
| MCP | 支持 MCP 协议 |
| 并发 | 最多20个子Agent并发 |

```bash
npm install -g codewhale
codewhale --model auto
```

**通过 SiliconFlow 或 OpenRouter 可以免费使用开源模型替代 DeepSeek。**

## ⚠️ 注意事项

1. **免费API限速**: 大部分免费API有 RPM/TPM 限制，不适合高频生产环境
2. **模型版本**: 免费层可能限制使用最新模型
3. **数据隐私**: 免费层数据可能被用于模型训练（需阅读各平台ToS）
4. **SLA保障**: 免费服务通常无SLA保障
5. **政策变化**: 免费政策随时可能调整
