## 关键发现
1. Oracle Cloud Always Free 仍然是价值最高的免费计算资源 (4 ARM / 24GB)
2. AWS 于 2025年7月重构了免费层，从12个月试用改为$200信用模式
3. Cloudflare 生态(Workers+R2+Pages+D1)是最全面的无信用卡免费方案
4. Turso 提供 9GB SQLite 边缘数据库，是存储量最大的免费数据库
5. 国内七牛云 10GB 对象存储永久免费，零信用卡
6. Reddit 社区推荐的零成本组合: Oracle + Cloudflare + Supabase + Grafana + GitHub Actions
